# Introdução ao HTML

Para entender como o HTML funciona precisamos entender como funciona o elemento. 

O elemento é a base do HTML, tudo no arquivo html é um elemento.

O elemento começa com uma tag de abertura, diz qual o tipo de elemento, a tag pode ter um atributo, depois temos o conteúdo e por ultimo fechamos a tag.

## Estrutura básica:

- !DOCTYPE html : Não é um elemento, apenas diz ao navegador o que estou escrevendo. 
- html: tudo que diz respeito ao conteúdo html precisa estar dentro desta tag.
- head: dentro deste elemento ficam meta informações, informações que o navegar precisa.
- meta: pode ser usada para  navegar fazer o entroud para os caracteres.
- title - /title: titulo da aba do navegador.
- /head: fechamento do head.
- body: onde fica todo o conteúdo da pagina.
- /body: fechamento body.
- /html: fechamento html. 

## Semântica:

Semântica é o estudo do significado e interpretação de palavras. Aplicando isso ao HTML, significa a melhor compreensão do HTML para o computador.

- section = seção genérica de conteúdo.
- header =  cabeçalho da página.
- article = conteúdo relevante dentro da página.
- aside = conteúdo relacionado ao conteúdo principal da página.
- footer = rodapé da página ou do conteúdo da página.
- h1 até h6 = Importância do titulo dentro de uma página, tamanho.  Só pode ter um h1 por página.

## Textos e links:

- h1 = Título da página. 
- h2 = Título de seção. 
- h3 = Título de artigo.
- p = Conteúdo do artigo = representa paragrafo, que pode ser adicionado fotos, vídeos e vários outros conteúdos. 

### Tags para links:

- a = utilizado para linkar página.

exemplo: 

a href="link">nome da página</a> = link da página que é para ser direcionado. 

a target="_blanck">link</a> = como é para essa página abrir(neste exemplo em uma outra aba do navegador.)

## Tag IMG

- img src =""> = é obrigatório, guarda o endereço de hospedagem da foto 
- img alt=""> = não é obrigatório mas muito recomendado pois é onde fica a descrição da img quando ela não é carregada ou para deficientes visuais. 

## tags li, ul e ol

### Como organizar listas com HTML:

- ul> = lista que a ordem não é importante
- ol>= lista que a ordem é muito importante, ficam identificados com número ou algarismos romanos.
- li> = item da lista
